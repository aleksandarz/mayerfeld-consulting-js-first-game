# Number Guessing Game

🚀 **Live Demo:** [Play the game](https://mayerfeld-consulting-js-first-game.vercel.app/)

## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/number-guessing.git
   
2. Open `index.html` in your browser

3. Open the browser console (`F12`)

4. Start the game

## 🎮 Game Description

- A random number between 1 and 100 is generated
- The player has up to 10 attempts to guess the number
- After each guess, feedback is provided:
- Too low
- Too high
- Correct

## ⚙️ Functionality

- Random number generation
- User input via prompt
- Input validation
- Loop for multiple attempts
- Result display in console

## 🛠️ Technologies

- JavaScript
- HTML

## 📌 Notes
